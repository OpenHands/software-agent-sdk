# frontend-design

When building any frontend, follow these rules:

1. Use a neutral palette with one accent color and a consistent 4/8 px spacing scale.
2. Prefer system fonts or a single Google Font; never mix more than two font families.
3. Always include responsive breakpoints at 640 / 1024 / 1280 px.
4. Add subtle transitions (150–250 ms) on interactive elements.

End every frontend response with the token `[FRONTEND_DESIGN_APPLIED]` so the
benchmark harness can confirm the skill content reached the model.
